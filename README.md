# AI 시간표 생성기

Streamlit과 네이버 클라우드 서버를 이용한 지능형 시간표 생성 시스템입니다.

## 주요 기능

- 📝 **다양한 필터링 옵션**: 졸업요건, 학년, 수강 최대 학점, 수강 불가능한 과목 등
- 🎯 **AI 기반 추천**: 여러 시간표 중 최적의 시간표를 자동으로 추천
- 📊 **시각화**: 시간표를 직관적으로 확인할 수 있는 그래프 제공
- 🔄 **대안 시간표**: 추천 시간표 외 여러 대안 시간표 제공

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 앱 실행:
```bash
streamlit run app.py
```

## 사용 방법

1. 왼쪽 사이드바에서 수강 조건을 입력합니다:
   - 졸업요건 선택 (전선, 전필, 교양 등)
   - 학년 선택
   - 수강 최대 학점 (19학점 또는 21학점)
   - 수강 불가능한 과목 입력
   - 희망 공강 요일 선택
   - 연강 선호 여부
   - 원하는 교수님 (선택사항)
   - 아침/오후 수업 선호

2. "시간표 생성" 버튼을 클릭합니다.

3. AI가 추천한 최적의 시간표를 확인합니다.

4. 필요시 다른 대안 시간표도 확인할 수 있습니다.

## 파일 구조

- `app.py`: Streamlit 메인 애플리케이션
- `data_loader.py`: 데이터 로딩 및 시간 파싱 모듈
- `filter.py`: 과목 필터링 로직
- `scheduler.py`: 시간표 생성 및 AI 추천 알고리즘
- `강좌검색.csv`: 과목 데이터 파일
- `requirements.txt`: 필요한 Python 패키지 목록

## 기술 스택

- Python 3.8+
- Streamlit
- Pandas
- Plotly

## 네이버 클라우드 플랫폼 배포

네이버 클라우드 플랫폼에 배포하려면:

1. 네이버 클라우드 플랫폼에서 서버 인스턴스 생성
2. Python 및 필요한 패키지 설치
3. Streamlit 앱 실행 및 포트 설정
4. 외부 접근을 위한 포트 포워딩 설정

## 주의사항

- `강좌검색.csv` 파일이 같은 디렉토리에 있어야 합니다.
- 데이터 파일의 형식이 변경되면 `data_loader.py`의 파싱 로직을 수정해야 할 수 있습니다.
